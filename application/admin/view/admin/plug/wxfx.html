    • 使用说明

    1.请先进入微信公众平台申请微信公众号。

    2.如需使用高级功能，请按照微信公众平台提示，进行公众号认证，查看认证教程。

    3.请将申请好的公众号信息，按照提示填入对应选项，进行验证。

    4.验证成功之后，即可配置微网站的其他功能。

    注：此功能需网站进行ICP备案及付费配置才可以使用。案例：点击查看
